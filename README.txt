Este é um jogo independente desenvolvido em Python.
Se o Windows bloquear a execução:

1. Clique com botão direito no arquivo .exe
2. Selecione "Propriedades"
3. Marque a opção "Desbloquear" (se disponível)
4. Clique em "Aplicar" e depois em "OK"

O jogo é seguro e não contém malware.
